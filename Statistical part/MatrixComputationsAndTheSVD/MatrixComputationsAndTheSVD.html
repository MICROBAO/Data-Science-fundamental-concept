Positive Definite and Positive Semidefinite Matrices

"Positive semidefinite": all eigenvalues of \(A\) are non-negative real values.

"Positive definite": all eigenvalues of \(A\) are positive real values.

The determinant of a matrix \(A\) is the product of its eigenvalues.

Any real matrix \(A\) of the form \(A = X' X\) is real symmetric and positive semidefinite.

Interpreting the Eigendecomposition Geometrically¶

In [17]: 
%%R -h 400 -w 800

LinearTransformPlot = function (A = matrix(c(9/5,2/5,2/5,3/5),2,2), n = 60) {

#   The 2x2 matrix A should be real and symmetric;
#   it is used to map the unit circle (with a total of n points) to an ellipse;
#   the axes of the ellipse are the eigenvectors of A, stretched by its eigenvalues.

   t = seq(0, 2*pi, length = n)

   x = rbind( cos(t) , sin(t) )   #   x  =  a sequence of n vectors (points on the unit circle)
   Ax = A %*% x;                  #  Ax  =  the result of transforming all vectors in x by A

   x1 = x[1,];
   x2 = x[2,];

   Ax1 = Ax[1,];
   Ax2 = Ax[2,];

   A_eigen = eigen(A)
   lambda = A_eigen$values
   L      = diag(A_eigen$values)
   Q      = A_eigen$vectors

   e1 = Q[,1]  #  1st eigenvector
   e2 = Q[,2]  #  2nd eigenvector

   if (max(e1) <= 0) {
      e1 = -e1; Q[,1] = -Q[,1];  # if e is an eigenvector, then -e is an eigenvector too:
      e2 = -e2; Q[,2] = -Q[,2];  # so we choose e1 so that it has at least one positive entry.
   }

   xmax = max( lambda[1], 1.0 ) * 1.33
   ymax = max( lambda[2], 1.0 ) * 1.33

   QL = Q %*% L
   Id = diag(c(1,1))
   zeroes = c(0,0)

   row_format = " ( % 4.2f  % 4.2f ) "
   A_format = paste(" A\n ", row_format, "\n", row_format)
   A_values = sprintf( A_format, A[1,1],A[1,2], A[2,1],A[2,2] )
   QL_format = paste(" Q                       L\n ", row_format, row_format, "\n", row_format, row_format)
   QL_values = sprintf(QL_format, Q[1,1],Q[1,2],L[1,1],L[1,2], Q[2,1],Q[2,2],L[2,1],L[2,2])

   opar = par(mfrow=c(1,2), cex=0.85, cex.main=1.0, cex.sub=1.0, font.sub=2)  # make a 1x2 set of plots in one figure

   plot( x1, x2, col="red", pch=20,
      asp=1.0, xlim = c(-xmax, xmax), ylim = c(-ymax,ymax), xlab="", ylab="",
      main="transformation by A of vectors on the unit circle", sub=A_values, col.sub="blue" )
   arrows( x1, x2, Ax1, Ax2, col="green", lwd=1.0, angle=8, length=0.1 )
   arrows( zeroes, zeroes,  Q[1,],  Q[2,], col="red",  lwd=2, angle=8, length=0.1 )
   arrows( zeroes, zeroes, QL[1,], QL[2,], col="blue", lwd=2, angle=8, length=0.1 )
   points( Ax1, Ax2, col="blue", pch=20)

   plot( x1, x2, col="red", pch=20,
      asp=1.0, xlim = c(-xmax, xmax), ylim = c(-ymax,ymax), xlab="", ylab="",
      main="explanation of the transformation by A", sub=QL_values, col.sub="blue" )
   points( Ax1, Ax2, col="blue", pch=20)
   arrows( zeroes, zeroes,  Q[1,],  Q[2,], col="red",  lwd=2, angle=8, length=0.1 )
   arrows( zeroes, zeroes, QL[1,], QL[2,], col="blue", lwd=2, angle=8, length=0.1 )
   text( 1.25 *  Q[1,], 1.25 *  Q[2,], c(expression(e[1]), expression(e[2])) )
   text( 1.25 * QL[1,], 1.25 * QL[2,], c(expression(lambda[1]*e[1]), expression(lambda[2]*e[2])) )

   par(opar)
}

The Singular Value Decomposition (SVD)

Theorem: If \(A\) is an arbitrary (real or complex) \(n \times p\) matrix where \(n > p\), there is an SVD matrix decomposition \[ A = U S V'\] where \(U\) is an \(n \times n\) unitary matrix, \(V\) is a \(p \times p\) unitary matrix, and \(S\) is an \(n \times p\) matrix whose nonzero entries arise only on its diagonal.

%%R -h 400 -w 1000

LinearTransformPlot = function (A = matrix(c(1.4, 1.3, 0.1, 0.7), 2,2), n = 60) {

#   The 2x2 matrix A should be real and symmetric;
#   it is used to map the unit circle (with a total of n points) to an ellipse;
#   the axes of the ellipse are the eigenvectors of A, stretched by its eigenvalues.
    
   t = seq(0, 2*pi, length=n)
   x1 = cos(t)
   x2 = sin(t)
   x = rbind( x1,  x2 )     #  x  = the unit circle  (= 2 x n matrix of (cos(t),sin(t)) vectors)

   Ax = A %*% x             #  Ax = the transformation by A of every vector in x
   Ax1 = Ax[1,]
   Ax2 = Ax[2,]

   USV = svd(A)
   sigma = USV$d
   S = diag(sigma)
   U = USV$u
   V = t(USV$v)

   u1 = U[,1]  # 1st left eigenvector
   u2 = U[,2]  # 2nd left eigenvector
   v1 = V[,1]  # 1st right eigenvector
   v2 = V[,2]  # 2nd right eigenvector
   if (max(u1) <= 0) {
      u1 = -u1; U[,1] = -U[,1];  # if u1 is an eigenvector, then -u2 is an eigenvector too:
      v1 = -v1; V[,1] = -V[,1];  # so choose u1 so that it has at least one positive entry.
   }
   if (max(u2) <= 0) {
      u2 = -u2; U[,2] = -U[,2];  # if u2 is an eigenvector, then -u2 is an eigenvector too:
      v2 = -v2; V[,2] = -V[,2];  # so choose u2 so that it has at least one positive entry.
   }

   U_sigma = U %*% diag(sigma)
   xmax = max( sigma[1], 1.0 ) * 1.33
   ymax = max( sigma[2], 1.0 ) * 1.33
   zeroes = c(0, 0)

   row_format = " ( % 4.2f  % 4.2f ) "
   A_format = paste("A\n ", row_format, "\n", row_format)
   A_values = sprintf( A_format, A[1,1],A[1,2], A[2,1],A[2,2] )
   USV_format = paste("U                       S                       V\n ",
       row_format, row_format, row_format, "\n", row_format, row_format, row_format)
   USV_values = sprintf(USV_format, U[1,1],U[1,2],S[1,1],S[1,2],V[1,1],V[1,2], U[2,1],U[2,2],S[2,1],S[2,2],V[2,1],V[2,2])

   opar = par(mfrow=c(1,2), cex=0.85, cex.main=1.0, cex.sub=1.0, font.sub=2 )   #  plot 2 figures in a 1x2 arrangement

   plot( x1, x2, pch=20, col="red",
      asp=1.0, xlim = c(-xmax, xmax), ylim = c(-ymax,ymax), xlab="", ylab="",
      main = "transformation by A of vectors on the unit circle",
      sub  = A_values, col.sub="blue" )
   arrows( x1, x2, Ax1, Ax2, col="green", lwd=1, angle=8, length=0.1 )
   points( Ax1, Ax2, pch=20, col="blue" )

   plot( x1, x2, pch=20, col="red",

The Economy version of the SVD

Theorem: If \(A\) is an arbitrary (real or complex) \(n \times p\) matrix where \(n > p\), there is an SVD matrix decomposition \[ A = U S V'\] where \(U\) is an \(n \times p\) prefix of a unitary matrix, \(V\) is a \(p \times p\) unitary matrix, and \(S\) is an \(p \times p\) matrix whose nonzero entries arise only on its diagonal.

By default, R computes this "economy" version of the SVD. To obtain the full SVD, you must ask it to compute all eigenvectors in \(U\).

Matrix Approximation with the SVD

Suppose \(A\) is an \(n \times p\) matrix where \(n > p\), and we have the SVD matrix decomposition $ A = U S V'.$ Assume that S = diag( 1, , p )$ is an \(n \times p\) matrix in which the only the first \(k\) values on the diagonal are nonzero.

Then \(A = U^{(k)} S^{(k)} V^{(k)}'\), where \(U^{(k)}\) is an \(n \times k\) matrix containing the first \(k\) columns of \(A\), \(S^{(k)}\) is a \(k \times k\) matrix containing the upper left \(k\times k\) submatrix of \(S\), and \(V^{(k)}\) is an \(n \times k\) matrix containing the first \(k\) columns of \(V\).

svd_approx = function(A, k) {
    Asvd = svd(A, nu=k, nv=k)
