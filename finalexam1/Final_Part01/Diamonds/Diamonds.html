Final Exam Dataset #1: Diamond Price Prediction¶

This is a dataset with information on about 60 thousand diamonds.

Your goal is to develop a model that takes attributes of a diamond as input, and produces a prediction about the price of the diamond as output.

Specifically: develop a model that predicts WHETHER THE PRICE OF A GIVEN DIAMOND WILL BE GREATER THAN $1500. (Your prediction will be 0 if the price is below $1500, and 1 if the price is above $1500.)

About 20 thousand have a price below $1500, and the remaining diamonds have a price above. Thus a simple model that always predicts high prices might be right about 2/3 of the time. Your job is to do better than this baseline rate.

More accurate statistics about columns of the dataset are included at the end of this notebook.

Schema of the Dataset¶

The dataset has the following columns:

Caratweight of the diamond in carats, rounded to an integer (1 carat = 0.2 grams)
Cutquality of the cut {Fair, Good, VeryGood, Premium, Ideal}
Colorcolor code: { D > E > F > G > H > I > J } (J is worst, D is best)
Clarityclarity code: { I1 < SI1 < SI2 < VS1 < VS2 < VVS1 < VVS2 < IF } (I1 is worst, IF is best)
xLength in mm (numeric value between 0 and 6)
yWidth in mm (numeric value between 0 and 9)
zDepth in mm (numeric value between 0 and 6)
Depthtotal depth percentage = 2*z/(x+y)
Tablewidth of top of diamond relative to widest point
Pricein US dollars

Caution¶

The datset has skewed distributions. Please check below.
